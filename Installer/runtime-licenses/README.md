# Runtime Bundle License Notices

This directory is copied into `WAPL-Runtime-v*.zip` as `LICENSES/`.

The runtime bundle includes WebApp Launcher project notices plus license files
for the bundled runtime/tool distributions:

- `WebAppLauncher/`: WebApp Launcher Apache-2.0 license and third-party notices.
- `Python/`: Python 3.13/3.14 license files, bundled pip license files, and Tk license terms.
- `Node.js/`: Node.js, npm, and Corepack license files for Node.js LTS 22/24.
- `Git/`: Git for Windows license files and bundled MSYS2/MinGW license trees.
- `pnpm/`: pnpm license and license files from the bundled pnpm distribution dependencies.
- `uv/`: uv MIT and Apache-2.0 license files.

The original runtime and tool directories also retain their upstream license
files in place. This folder exists to provide a single predictable notice
location for the launcher UI and runtime bundle distribution.
